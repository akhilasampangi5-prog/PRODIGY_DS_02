# Task-02: Data Cleaning and EDA

## Objective
Performed data cleaning and exploratory data analysis on Titanic dataset.

## Steps Performed
- Checked missing values
- Filled Age with median
- Filled Embarked with mode
- Dropped Cabin column
- Created visualizations

## Patterns Found
- Females survived more.
- First class passengers had higher survival.
- Fare positively correlated with survival.
- Gender is most influential factor.

## Tools Used
Python, Pandas, Matplotlib, Seaborn
